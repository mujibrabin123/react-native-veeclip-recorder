# @talkvee/react-native-veeclip-recorder

Native duet clip recorder for Talkvee

## Installation


```sh
npm install @talkvee/react-native-veeclip-recorder
```


## Usage


```js
import { multiply } from '@talkvee/react-native-veeclip-recorder';

// ...

const result = multiply(3, 7);
```
